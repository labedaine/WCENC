Réglement

Vous pronostiquez pour chaque match de la compétition et engrangez des points en fonction des résultats :


 Barème de points pour les matchs de Groupe
  • 1 point si le bon résultat est trouvé (équipe vainqueur ou match nul)
  • 1 point supplémentaire si l'écart de but entre les deux équipes est correct
  • 1 point supplémentaire pour le bon nombre de buts inscrit par équipe (score exacte)
 Barème de points pour les Huitièmes de finale
  • 2 points si le bon résultat est trouvé (équipe vainqueur ou match nul à l'issue du temps réglementaire )
  • 2 points supplémentaires si l'écart de but entre les deux équipes est correct (à l'issue du temps réglementaire )
  • 2 points supplémentaires pour le bon nombre de buts inscrit par équipe (score exacte à l'issue du temps réglementaire )
 Barème de points pour les Quarts de finale
  • 3 points si le bon résultat est trouvé (équipe vainqueur ou match nul à l'issue du temps réglementaire )
  • 3 points supplémentaires si l'écart de but entre les deux équipes est correct (à l'issue du temps réglementaire )
  • 3 points supplémentaires pour le bon nombre de buts inscrit par équipe (score exacte à l'issue du temps réglementaire )
 Barème de points pour les Demi-finales
  • 4 points si le bon résultat est trouvé (équipe vainqueur ou match nul à l'issue du temps réglementaire )
  • 4 points supplémentaires si l'écart de but entre les deux équipes est correct (à l'issue du temps réglementaire )
  • 4 points supplémentaires pour le bon nombre de buts inscrit par équipe (score exacte à l'issue du temps réglementaire )
 Barème de points pour la Petite finale (Si elle existe)
  • 5 points si le bon résultat est trouvé (équipe vainqueur ou match nul à l'issue du temps réglementaire )
  • 5 points supplémentaires si l'écart de but entre les deux équipes est correct (à l'issue du temps réglementaire )
  • 5 points supplémentaires pour le bon nombre de buts inscrit par équipe (score exacte à l'issue du temps réglementaire )
 Barème de points pour la Finale
  • 6 points si le bon résultat est trouvé (équipe vainqueur ou match nul à l'issue du temps réglementaire )
  • 6 points supplémentaires si l'écart de but entre les deux équipes est correct (à l'issue du temps réglementaire )
  • 6 points supplémentaires pour le bon nombre de buts inscrit par équipe (score exacte à l'issue du temps réglementaire )
