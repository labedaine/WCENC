Reglement

Vous pronostiquez pour chaque match de la compétition et engrangez des points en fonction des résultats :


 Barème de points pour les matchs de Groupe
  • 1 point si le bon résultat est trouvé (équipe vainqueur ou match nul)
  • 1 point si l'écart de but entre les deux équipes est correct
  • 1 point pour le bon nombre de buts inscrit par équipe
 Barème de points pour les Huitièmes de finale
  • 4 points si le bon résultat est trouvé (équipe vainqueur ou match nul à l'issue d'éventuelles prolongations)
  • 4 points si l'écart de but entre les deux équipes est correct (à l'issue d'éventuelles prolongations)
  • 4 points pour le bon nombre de buts inscrit par équipe (à l'issue d'éventuelles prolongations)
 Barème de points pour les Quarts de finale
  • 5 points si le bon résultat est trouvé (équipe vainqueur ou match nul à l'issue d'éventuelles prolongations)
  • 5 points si l'écart de but entre les deux équipes est correct (à l'issue d'éventuelles prolongations)
  • 5 points pour le bon nombre de buts inscrit par équipe (à l'issue d'éventuelles prolongations)
 Barème de points pour les Demi-finales
  • 6 points si le bon résultat est trouvé (équipe vainqueur ou match nul à l'issue d'éventuelles prolongations)
  • 6 points si l'écart de but entre les deux équipes est correct (à l'issue d'éventuelles prolongations)
  • 6 points pour le bon nombre de buts inscrit par équipe (à l'issue d'éventuelles prolongations)
 Barème de points pour la Petite finale
  • 7 points si le bon résultat est trouvé (équipe vainqueur ou match nul à l'issue d'éventuelles prolongations)
  • 7 points si l'écart de but entre les deux équipes est correct (à l'issue d'éventuelles prolongations)
  • 7 points pour le bon nombre de buts inscrit par équipe (à l'issue d'éventuelles prolongations)
 Barème de points pour la Finale
  • 8 points si le bon résultat est trouvé (équipe vainqueur ou match nul à l'issue d'éventuelles prolongations)
  • 8 points si l'écart de but entre les deux équipes est correct (à l'issue d'éventuelles prolongations)
  • 8 points pour le bon nombre de buts inscrit par équipe (à l'issue d'éventuelles prolongations)
